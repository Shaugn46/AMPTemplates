# AMPTemplates

Private AMP Generic templates used by my AMP controller.

## Stardew Valley Dedicated Server

This repository contains a private Windows Generic template for JunimoServer/Stardew Valley:

- `stardew-private.kvp`
- `stardew-privateconfig.json`
- `stardew-privateports.json`

The template expects the Windows target node to have the Stardew server repo and host scripts at `D:\Stardew\server`.
